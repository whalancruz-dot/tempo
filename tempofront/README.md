🚀 Guia de Inicialização - Frontend (Angular)

Este guia contém os comandos necessários para instalar as dependências e colocar a interface do sistema em execução.



📋 Pré-requisitos

Antes de começar, certifique-se de ter instalado:

    Node.js (LTS recomendada)

    Angular CLI (npm install -g @angular/cli)

🛠️ Passo a Passo

1. Instalação de Dependências

Abra o terminal na pasta raiz do projeto frontend e execute o comando abaixo para baixar todas as bibliotecas necessárias (pasta node_modules):
CMD: npm i

2. Execução da Aplicação

Após a instalação concluir, inicie o servidor de desenvolvimento:
CMD: npm start
